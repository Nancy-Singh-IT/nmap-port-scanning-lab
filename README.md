# Nmap Port Scanning Lab

## Objective
Learn basic network scanning using Nmap.

## Tools Used
- Ubuntu VM
- Nmap
- Python HTTP Server

## What I Did
- Found local IP using `ip a`
- Started HTTP server on port 8000
- Scanned open ports using Nmap

## Result
- Port 8000 (HTTP) was open
- Service detected: SimpleHTTPServer

## Learning Outcome
- Understood basic port scanning
- Learned how services run on ports
- Practiced basic cybersecurity recon
